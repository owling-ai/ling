# Ling 定制音色可行性与实施方案

> 日期：2026-07-11
>
> 详细实现与验证记录见 [`docs/custom-voice.md`](./docs/custom-voice.md)。

## 决策

旧版四个 Gemini 角色音色已经撤回。严格评审确认它们都保留成年人声带和声道底色，角色提示只能改变表演方式，不能产生真实儿童听感；音高和共振处理也会带来变调与卡通化伪影。

当前采用两档火山 `seed-tts-2.0` 原生童声：

- `sunny` / 小晴天 / `zh_male_tiancaitongsheng_uranus_bigtts`；
- `sprout` / 小青芽 / `ICL_uranus_zh_female_jiaxiaozi_tob`。

两档都通过实际 ByteRTC 录回样本、普通/安慰/兴奋三类台词和中英混说评审。最终儿童听感为 `8/10`，成人装嫩风险为 `2/10`，长聊舒适度为 `8/10`。疑似现成卡通角色仿声、成年夹嗓和不稳定候选均已排除。

## 架构变化

```text
浏览器 -> ByteRTC ASR/VAD/打断/视频
       -> Ling 鉴权 SSE 回调
       -> Gemini 文本模型
       -> seed-tts-2.0 原生童声
       -> ByteRTC 下行
```

这保留 Gemini API、流式回复、语音打断和视频理解，但不再是 Gemini Live 原生音频到音频链路。三次真实测试从服务端触发到完整回复字幕为 `2.2-3.7s`；一次无视频测试的首个非静音下行音频约 `2.58s`。延迟和故障面均高于原生 Live，后续仍需扩大首音频帧样本并继续优化。

## 合规边界

- 不使用真人儿童录音，不冒充真实儿童；
- 不复制名人、动画角色、游戏角色或无权使用的第三方声音；
- 不把火山公版 voice 重新克隆到 Gemini；
- `replicatedVoiceConfig` 继续保持实验状态，只有取得成年声音提供者合法参考和授权后才能验证；
- 参考录音、授权录音和授权签名不得进入 Git、日志或前端存储。

## 配置

```bash
GEMINI_API_KEY=...
VOLCENGINE_RTC_APP_ID=...
VOLCENGINE_RTC_APP_KEY=...
VOLCENGINE_ACCESS_KEY=...
VOLCENGINE_SECRET_KEY=...
LING_VOLC_GEMINI_LLM_URL=https://ling.example.com/integrations/volcengine/gemini
LING_VOLC_GEMINI_MODEL=gemini-3.1-flash-lite
LING_VOLC_VOICE_PROFILE=sunny
LING_REALTIME_PROVIDER=volcengine
```

`LING_VOLC_GEMINI_LLM_URL` 必须是公网 HTTPS 地址并指向当前 Ling 进程。Gemini API Key 只保留在后端；火山只获得单进程、临时生成的回调 Bearer Token。
