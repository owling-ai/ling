# 灵灵世界线视频提示词

每个文件都是一份可以独立复制使用的完整提示词，已经包含：

- 角色一致性约束；
- 5-7 秒场景动作；
- 9:16 全屏构图和 UI 安全区；
- 统一视觉风格；
- 负面提示词。

## 建议流程

1. 先使用 [00-角色定妆基准.md](./00-角色定妆基准.md) 生成一张干净的角色定妆图。
2. 后续所有视频任务都上传同一张定妆图作为角色参考。
3. 每次只复制一个世界线文件中的提示词，生成一个 5-7 秒视频。
4. 先固定角色和动作，再为同一场景生成 2-3 个不同镜头变体。
